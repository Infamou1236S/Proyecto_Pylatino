El Sistema de Registro de Hábitos es una aplicación diseñada para ayudar al usuario a crear y mantener hábitos diarios de manera sencilla y agradable.
Permite:
-Registrar nuevos hábitos.
-Marcar si se cumplieron o no cada día.
-Consultar un historial completo de todos los hábitos registrados.
-Visualizar un resumen semanal usando pandas.
-Exportar los datos a un DataFrame para análisis posteriores.
